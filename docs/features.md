# Features

AutoInsight comes packed with features that demonstrate autonomous AI economics.

## Report Generation

### AI-Powered Content

- Generates ~500-word research reports on fintech/Web3 topics
- Covers topics: DeFi, RWA, Stablecoins, AI x Crypto, Bitcoin Ecosystem, Cross-border Payments
- Powered by Anthropic Claude API

### Preview System

- Free preview (~100 words + teaser) for all visitors
- Full report unlocks after $0.50 USDC payment
- Clear call-to-action for payment

## Micropayments with Locus

### Payment Integration

- Locus Checkout SDK integration for $0.50 USDC payments
- Instant report unlock on payment confirmation
- Transaction verification before content release

### Supported Features

- USDC currency support
- Instant confirmation
- Transaction hash tracking

## Autonomous Agent Loop

### Automatic Operation

The backend agent performs these steps autonomously:

1. **Check Wallet Balance**: Verify available funds
2. **Fetch Recent Earnings**: Get reader payments from Locus
3. **Select APIs**: Choose 3-5 data APIs based on current topic
4. **Purchase APIs**: Pay for data from Locus catalog
5. **Generate Report**: Create new report with Claude using purchased data
6. **Update State**: Refresh dashboard metrics

### Demo Control

- "Advance to Next Week" button for demonstration purposes
- Real-time activity logging
- Step-by-step visualization

## Dashboard & Transparency

### Financial Metrics

- **Total Earnings**: Sum of all reader payments
- **Total Spent**: Sum of all API purchases and AI calls
- **Current Balance**: Current wallet balance
- **Net Profit/Loss**: Earnings minus spending

### Transaction History

- Complete log of all earnings (payments received)
- Complete log of all spending (APIs purchased)
- Timestamps and amounts for each transaction

### Visualizations

- **Earnings Chart**: Bar chart showing earnings over weeks
- **Spending Chart**: Bar chart showing spending over weeks
- **Earn vs Spend**: Combined visualization comparing income vs expenses

### Agent Status

- Real-time status updates
- Current activity display
- Last action timestamp

### Spending Controls

- Max weekly budget slider
- Spending policy configuration
- Budget limit enforcement
